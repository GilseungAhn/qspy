## qspy: 파이썬을 이용한 퀀트 투자 전략 검증

### 설치
`pip install git+https://github.com/GilseungAhn/qspy.git`

### 사용 방법
[위키 참고](https://github.com/GilseungAhn/qspy/wiki/qspy-%EC%82%AC%EC%9A%A9-%EC%84%A4%EB%AA%85%EC%84%9C)

### 주요 기능
* 금융 데이터 다운로드
* 캔들 패턴 기반 투자 전략 검증
* 재무 지표 기반 투자 전략 검증 (추가 예정)
